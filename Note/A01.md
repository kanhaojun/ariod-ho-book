## Code

兩段程式分為 setup()、 loop()，前者為一開始即做一次的 setup()，而後者 loop() 為重複做的事情。

```

void setup() {

// put your setup code here, to run once:

// 開始只需要做一次的

}

void loop() {

// put your main code here, to run repeatedly:

//重複做的事情

}

```
